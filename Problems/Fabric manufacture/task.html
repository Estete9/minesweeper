Fabric manufacture

Complete the secondary constructor of the given class Fabric. The secondary constructor should take color, pattern, and patternColor as arguments and change the corresponding property values. Then, updated information about the fabric should be printed.

The complete process of creating an object of the Fabric class should be as following:

  1. The fabric of the default grey color is created.
  2. The fabric is dyed in some other color.
  3. Some color pattern is applied to the fabric.

Each step is accompanied by the output:

  1. "grey fabric is created" — for initial object creation, as shown in the primary constructor in the template;
  2. "the fabric is dyed <color>" — for dying fabric in a different color, where <color> is the corresponding argument, provided to the secondary constructor;
  3. "<patternColor> <pattern> pattern is printed on the fabric" — for applying a color pattern on the fabric, where <pattern> and <patternColor> are arguments, provided to the secondary constructor.

For example, creating a new object val fabric = Fabric("white", "dots", "black") is accompanied by the following output:

grey fabric is created
the fabric is dyed white
black dots pattern is printed on the fabric
