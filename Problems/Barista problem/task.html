Barista problem

Write the class EspressoMachine with the Float property costPerServing. Objects of the class can be created with two secondary constructors.

The first constructor takes arguments coffeeCapsulesCount and totalCost, which are of Int and Float type respectively. The value of the property costPerServing is calculated as the cost of one coffee capsule.

The second constructor takes properties coffeeBeansWeight and totalCost, both of the type Float. The value of the property costPerServing is calculated a little differently. Consider that for each cup of coffee, the machine takes 10.6 weight units of coffee beans. You need to calculate how many servings can be made of the given coffeeBeansWeight and then calculate the cost of one serving using totalCost of coffee beans.

Hint: be extra careful with the division when calculating the cost of coffee serving made with beans.
