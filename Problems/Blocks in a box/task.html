Blocks in a box

Let's create a simple class with a nested object. There is a box of children's blocks. All of them are the same size but can be different colors. We will store these objects in a class Block with one property color and a nested object DimProperties with properties length and width. Your task is to create this structure!

In addition, your nested object should have a blocksInBox() function that receives the length and width of a box and returns how many blocks this box can store.

Make the properties of a nested object mutable for the validation system! You may initialize them with any numbers. Don't worry about heights, solve it like a 2D problem. Don't rotate the blocks.



Memory limit: 256 MB
Time limit: 8 seconds
