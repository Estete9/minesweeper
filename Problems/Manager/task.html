Manager

You want to create a singleton Manager that can solve some tasks. Let's write a skeleton for this case.

First, you have a class Task with one field name. Second, you need to create a singleton Manager with a function solveTask() and an Int property solvedTask.

The function solveTask() receives an instance of the Task class and prints a line "Task NAME solved!", where NAME is the task's name.

Also, you should increase by one a solvedTask property each time you solve a task. Good luck!
